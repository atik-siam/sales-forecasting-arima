# Sales Forecasting Using ARIMA

## Objective
Forecast monthly retail sales using time series modeling for business planning.

## Dataset
Supermarket sales data aggregated to monthly frequency.

## Methodology
- Time series aggregation
- Stationarity testing (ADF)
- ARIMA(1,1,0) modeling
- Residual diagnostics
- 6-month forecast evaluation

## Results
- Forecast horizon: 6 months
- MAPE: ~36%

## Tools
Python, pandas, statsmodels, matplotlib
